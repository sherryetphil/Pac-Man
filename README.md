# 小精靈 Mini Game（Mac）

這是一個純前端的小精靈遊戲，不需要安裝套件。

## 在 Mac 上執行

1. 用 Finder 打開 `index.html`（雙擊即可）。
2. 或在 Terminal 執行：

```bash
cd /Users/mikibaby/Documents/codex2
python3 -m http.server 8000
```

接著在瀏覽器開啟：
`http://localhost:8000`

## 操作方式

- 方向鍵 或 `WASD`：移動
- `P`：暫停 / 繼續
- `R`：重新開始

## 遊戲規則

- 吃小豆子：+10 分
- 吃能量豆：+50 分，短時間無敵
- 吃到全部豆子即獲勝
- 被鬼碰到會扣生命，生命歸零則遊戲結束
